# EOSLT – UX style

Project-specific UX, branding, and design patterns. Update this file as the product evolves.

## Color palette

- **Background (light):** `#ffffff` (--background)
- **Foreground (light):** `#171717` (--foreground)
- **Background (dark):** `#0a0a0a`
- **Foreground (dark):** `#ededed`

## Typography

- **Sans:** Geist (--font-geist-sans)
- **Mono:** Geist Mono (--font-geist-mono)

## Spacing & layout

- Max content width: `max-w-5xl` (1024px)
- Main padding: `px-4 py-8 sm:px-6`

## Motion & accessibility

- Prefer reduced motion where appropriate.
- Ensure focus states and keyboard navigation for interactive elements.
